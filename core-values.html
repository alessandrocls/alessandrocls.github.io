My Core Values

  • Bebas, Canggih, Harmonis, Hiburan, Kerja Sama, Kreatif, Loyalitas, Musik, Tenang
  • Saya merasa saya itu easy-going. Bebas melakukan apa saja dalam pertemanan asal tidak keterlaluan. Karena sehari-hari mengutak-utik komputer, saya juga termasuk canggih.
